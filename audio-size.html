Calculating Audio Size

Formula for File Size

File size = Sample rate × Bit depth × Channels × Duration

Audio Types

  • Uncompressed audio (e.g., WAV, AIFF)
  • Compressed audio (e.g., MP3, AAC)

Comparing Compressed and Uncompressed Files

Uncompressed files are larger and better in quality. Compressed files are smaller and easier to transfer but may lose some quality.
