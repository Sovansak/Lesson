About Us

What is storage media?

Storage media are the components of a computer system that store data. Common examples include:

  • CDs, DVDs and Blu-ray discs
  • Magnetic hard disks
  • Magnetic tape (found in cassettes)
  • Portable flash drives
  • USB pens or sticks
  • Memory cards

What is a storage device?

A storage device reads and writes data to and from storage media. Examples include:

  • Internal hard drive
  • CD/DVD/Blu-ray drive
  • External hard drive
